"Write comprehensive pytest unit tests for the following Python functions
in a FastAPI web vulnerability scanner project:

2. validate_target_url(raw_url) — validates and sanitizes URLs, returns (url, error)
3. get_scan_summary_stats() — reads global state, returns severity counts and uptime

- Normal use cases (expected valid input)

- Invalid input (None, wrong types, blocked values)

Organize tests into classes per function. Add a docstring to each test."
